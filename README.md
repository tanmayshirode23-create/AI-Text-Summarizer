# AI Text Summarizer

A conversational text summarization system built using Hugging Face Transformers and FastAPI.

## Features

- Dialogue summarization
- T5 Transformer model
- FastAPI backend
- Responsive frontend
- Hugging Face deployment

## Tech Stack

- Python
- FastAPI
- Transformers
- PyTorch
- HTML/CSS
- Hugging Face Spaces
